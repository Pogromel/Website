What is WarehouseWorker?

WarehouseWorker is project for my Secound Year at University, It was my first time using Unreal Engine C++ mainly.

This Game tought me a lot about C++, Creating PostProcess effects and creating own HLSL Materials, that will be applied onto anything. As it was Graphic Module, it made me create many assets for this game in blender.

this game helped me improve my Blender skills, and tought me many new things. As this game is incomplete it tought be a lot of things, such as creating materials, Writing own HLSL, Creating movement fully with C++.

Gameplay:

WarehouseWorker is game about your first day at work on werhouse, As you finish the tutorial scene you will be limited time of 5 Minutes in order to complete order. Move Quick as it need to be delivered on time.
